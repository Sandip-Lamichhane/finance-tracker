# Finance Tracker

A modern, full-stack finance tracking application built with the MERN stack (MongoDB, Express.js, React, Node.js). Track every rupee and take control of your personal finances.

## 🚀 Features

- **Expense Tracking**: Log and categorize your daily expenses
- **Income Management**: Record and monitor income sources
- **Budget Planning**: Set and track budgets for different categories
- **Financial Insights**: Visualize spending patterns with charts and reports
- **User Authentication**: Secure login and user management
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## 🛠 Tech Stack

### Frontend
- **React 19** - Modern React with hooks and concurrent features
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **DaisyUI** - Component library for Tailwind CSS
- **React Router** - Client-side routing
- **Formik & Yup** - Form handling and validation

### Backend (Planned)
- **Node.js** - JavaScript runtime
- **Express.js** - Web application framework
- **MongoDB** - NoSQL database
- **JWT** - Authentication tokens

## 📋 Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- MongoDB (for backend)

## 🔧 Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd finance-tracker
   ```

2. **Install frontend dependencies**
   ```bash
   cd frontend
   npm install
   ```

3. **Install backend dependencies** (when implemented)
   ```bash
   cd ../backend
   npm install
   ```

## 🚀 Running the Application

### Frontend
```bash
cd frontend
npm run dev
```
The application will be available at `http://localhost:5173`

### Backend (when implemented)
```bash
cd backend
npm start
```

## 🎯 Available Scripts

### Frontend
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
